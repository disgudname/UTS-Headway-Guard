Feed Codes

Maps a stable code (used in /api/rss/stop_arrivals/{code} and /api/cap/stop_arrivals/{code}) to the current TransLoc stop ID(s). Edit the stop ID(s) here whenever TransLoc renumbers a stop — no redeploy needed, and any signage already pointed at the code will pick up the change immediately. Enter more than one ID, comma-separated (e.g. 40, 113), when TransLoc represents one physical stop as separate IDs per route — arrivals from all of them are merged into a single feed.
